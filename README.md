# Option pricing Black Scholes

## Overview
Calculation of the price of a European call option using the Black-Scholes formula.  

## Objective
To understand option pricing mechanics and the impact of the different parameters (volatility, strike, and risk-free rates) on European call options.

## Materials used
- Python  
- math 

## Key features
- Calculates d1 and d2 for Black-Scholes formula  
- Computes European call option price
